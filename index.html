Methodology


These graphics were created by Daniel González Cappa with the sole purpose of being an example of different tools of journalism, code and infographics.

All the graphs shown are based on the results of the general elections of the United Kingdom 2019.

The data was collected and scrapped from various official and news websites; and analyzed with Excel.

The tools used were Adobe Illustrator and Photoshop (for the simplest graphics and visual elements), HTML, CSS and Javascript; and QGis and D3.js for the maps.

Two maps were made with the same data: one basic and one interactive. The purpose is exclusively to show how maps can be made with different tools.

The graphics created with Illustrator were also processed with the ai2illustrator plugin, a tool that allows you to transform graphic elements into HTML code.
